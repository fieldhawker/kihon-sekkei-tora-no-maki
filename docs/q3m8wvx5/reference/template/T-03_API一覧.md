## 認証

| 項目 | 内容 |
|-----|-----|
| 認証ヘッダー | `Authorization: Bearer <JWT_TOKEN>` |
| トークン取得 | `POST /auth/login` で取得 |
| トークン有効期限 | 24時間（設計書に合わせて変更） |
| 認証不要のエンドポイント | ログイン・パスワードリセット・公開API |

## API一覧

### 認証系

| API-ID | メソッド | パス | 概要 | 認証 | 対応機能ID |
|-------|--------|-----|-----|-----|---------|
| API-001 | POST | `/auth/login` | ログイン（JWTトークン発行） | 不要 | F-001 |
| API-002 | POST | `/auth/logout` | ログアウト | 必要 | F-002 |
| API-003 | POST | `/auth/password/reset` | パスワードリセットメール送信 | 不要 | F-003 |
| API-004 | PUT | `/auth/password/reset/:token` | パスワード再設定 | 不要 | F-003 |

## 共通レスポンス形式

### 成功時

```json
{
  "status": "success",
  "data": { ... },
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 150
  }
}
```

### エラー時

```json
{
  "status": "error",
  "code": "VALIDATION_ERROR",
  "message": "入力値に誤りがあります",
  "errors": [
    {
      "field": "email",
      "message": "メールアドレスの形式が正しくありません"
    }
  ]
}
```

### HTTPステータスコード

| コード | 意味 | 使用場面 |
|------|-----|--------|
| 200 | OK | 取得・更新成功 |
| 201 | Created | 新規作成成功 |
| 204 | No Content | 削除成功 |
| 400 | Bad Request | バリデーションエラー |
| 401 | Unauthorized | 認証が必要・トークン無効 |
| 403 | Forbidden | 権限なし |
| 404 | Not Found | リソースが存在しない |
| 422 | Unprocessable Entity | 処理できない入力（業務ルールエラー） |
| 500 | Internal Server Error | サーバー内部エラー |
