ヒアリングチェックリスト
要件定義
業務フロー・ユーザー・機能・データ・非機能・セキュリティ・運用など8カテゴリ・40項目以上の確認漏れ防止チェックリスト。各項目に「確認済 / 未確認 / N/A」とメモ欄。全体進捗をパーセント表示し、Markdownレポートとしてエクスポート可能。
要件定義 ヒアリング 確認漏れ防止

ヒアリングシート＆要件起こしアシスト
要件定義
打ち合わせメモ・会話ログを貼り付けると、キーワードをハイライトしつつ「機能要件 / 非機能要件 / 制約 / 未確認事項」の4カテゴリに自動振り分け。カードを手動で編集・追加でき、要件定義書の骨格をMarkdownでエクスポート。
要件定義 口頭議事録 要件構造化

未確認事項・宿題管理ツール
要件定義
ヒアリング中の「あとで確認する」事項をリアルタイムで記録。ミーティング単位でグルーピングし、未対応 / 確認中 / 解決済でステータス管理。解決内容を記録して「言った言わない」を防止。議事録形式でMarkdownエクスポート。

要件スコープ境界線エディタ
要件定義
要件・機能・画面を「✅ IN（今回作る）/ ❌ OUT（今回は作らない）/ 🔮 将来検討」の3ゾーンにドラッグ＆ドロップで仕分け。OUTの理由を記録してスコープクリープを防止。変更履歴を自動保存。Markdown / CSV エクスポート対応。
