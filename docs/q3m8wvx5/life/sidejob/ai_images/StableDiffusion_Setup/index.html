Stable Diffusion 環境構築ガイド¶

スーパーPOP・LINEスタンプ向け商用画像生成¶

対象：スーパーのPOP素材・LINEスタンプの商用販売を目的とした素人向け実践ガイド

PART 1：環境構築¶

Stability Matrix を使う（推奨・初心者向け）¶

Python・Gitの知識不要で、AUTOMATIC1111 / ComfyUI を1クリックでインストールできる統合管理ツール。

インストール手順

  1. lykos.ai から StabilityMatrix.exe をダウンロード
  2. C:\StabilityMatrix\ などの日本語・スペースを含まないフォルダに置く
  3. 起動 → 利用規約に同意
  4. 「インストールするUIを選択」画面で AUTOMATIC1111 を選ぶ
  5. 自動でPython環境・依存ライブラリがインストールされる（15〜30分）
  6. 「Launch」ボタンでブラウザが開いたら完了

AUTOMATIC1111 vs ComfyUI の選択基準

AUTOMATIC1111 ComfyUI
難易度 初心者向け 中級者向け
UI フォーム入力型 ノードベース
拡張機能 豊富 豊富（異なるもの）
POP・スタンプ用途 ◎ 推奨 ○
処理の自動化 △ ◎

→ まずはAUTOMATIC1111から始める

推奨PCスペック¶

項目 最低 推奨
GPU NVIDIA GTX 1060 6GB RTX 3060 12GB以上
RAM 16GB 32GB
ストレージ SSD 50GB空き SSD 100GB以上
OS Windows 10 Windows 11

GPU VRAMが少ない場合

6GB以下の場合はAUTOMATIC1111起動時に
--medvram または --lowvram オプションを追加する。
webui-user.bat の set COMMANDLINE_ARGS= の行に追記する。

PART 2：モデル選定（商用利用可能なもののみ）¶

モデルの追加方法¶

モデルは後からいつでも追加できる。主に以下の2つの方法がある。

方法①：Stability Matrix の Model Browser から追加（推奨）¶

  1. Stability Matrix を起動
  2. 左メニューの「Model Browser」をクリック
  3. 検索バーにモデル名を入力（例：Anything XL）
  4. サムネイルをクリックして「Install」ボタンを押す
  5. ダウンロード完了後、AUTOMATIC1111 を再起動
  6. txt2img 画面左上の「Stable Diffusion checkpoint」ドロップダウンから選択して使用

方法②：Civitai から手動でダウンロードして配置¶

  1. civitai.com でモデルを検索
  2. ライセンス（Permissions）を確認してから「Download」
  3. ダウンロードした .safetensors ファイルを以下のフォルダに配置する
# Stability Matrix 経由でインストールした場合のパス例
C:\StabilityMatrix\Data\Packages\stable-diffusion-webui\models\Stable-diffusion\

# 直接インストールした場合
C:\stable-diffusion-webui\models\Stable-diffusion\
  1. AUTOMATIC1111 の画面右上「🔄（Refresh）」ボタンをクリック
  2. checkpoint ドロップダウンに新しいモデルが表示される

LoRA の追加方法¶

LoRA（スタイルや特定の画風を追加するファイル）は配置先が異なる：

# LoRA の配置先
models\Lora\    ← ここに .safetensors を置く

プロンプトでの呼び出し方： 

<lora:LoRAファイル名:0.8>
# 数値（0.1〜1.0）は適用強度。0.6〜0.8 が一般的

ControlNet モデルの追加¶

ControlNet 拡張をインストール後、モデルファイルを以下に配置：

extensions\sd-webui-controlnet\models\

Hugging Face の lllyasviel/ControlNet-v1-1 からダウンロードできる。

ライセンスの確認方法¶

モデルをダウンロードする前に必ず確認する：

  • Civitai のモデルページ右上「Permissions」欄
  • Sell image they generate にチェック → 商用利用可
  • Hugging Face のモデルページ
  • License: creativeml-openrail-m → 商用利用可
  • License: Apache 2.0 → 商用利用可
  • LoRAも個別確認が必要：ベースモデルが商用OKでもLoRAが商用NGなら全体としてNG

スーパーPOP向け推奨モデル¶

Model Browser / Civitai で迷わず見つける方法

名前で検索すると類似モデルが大量に出て迷う。
Civitai のモデルIDをURLに直接入力するのが確実。
ブラウザで https://civitai.com/models/モデルID を開いてダウンロードし、
models\Stable-diffusion\ に配置する（方法②）が最も確実。
Stability Matrix の Model Browser では検索後に「Sort: Most Downloaded」で絞り込み、
作者名とサムネイルを下記と照合して選ぶ。

1. Anything XL（最推奨）¶

  • 用途：食べ物・野菜・果物・キャラクターの可愛いイラスト
  • ライセンス：CreativeML Open RAIL++-M（商用利用可）
  • ベース：SDXL
  • 特徴：鮮やかな色彩、POP的な見た目に最も適している。食材・キャラクターの親しみやすい描写が得意
  • Civitai URL：https://civitai.com/models/9409
  • 作者：Yuno779　ダウンロード数：800万超（2026年時点で最多クラス）

2. Dark Sushi Mix（食品・生鮮系に強い）¶

  • 用途：食べ物のリアル寄りイラスト・半リアルPOP
  • ライセンス：CreativeML Open RAIL-M（商用利用可）
  • ベース：SD1.5
  • 特徴：食材の色艶の表現が特に優れている。「おいしそう」な見た目を作りやすい
  • Civitai URL：https://civitai.com/models/24779
  • 作者：Udon_Oliver　バージョンは 2.25D を選ぶ（最もダウンロード数が多いバージョン）

3. blue_pencil-XL（手描き風POPに）¶

  • 用途：手書き風・温かみのあるイラストPOP
  • ライセンス：CreativeML Open RAIL++-M（商用利用可）
  • ベース：SDXL
  • 特徴：鉛筆・水彩風の柔らかいタッチが得意。地域密着型のスーパーPOPに合う雰囲気
  • Civitai URL：https://civitai.com/models/119012
  • 作者：bluepen5805　バージョンは v7.0.0（最新版）を選ぶ

4. Juggernaut XL（写真風・高品質リアル）¶

  • 用途：商品写真風の高品質素材
  • ライセンス：CreativeML Open RAIL++-M（商用利用可）
  • ベース：SDXL
  • 特徴：写真に近いリアルな質感。生鮮食品・惣菜のフォト風POPに使える
  • Civitai URL：https://civitai.com/models/133005
  • 作者：KandooAI　バージョンは Ragnarok （最新版）を選ぶ

LINEスタンプ向け推奨モデル¶

1. anima_pencil-XL（スタンプ最推奨）¶

  • 用途：表情豊かなキャラクタースタンプ
  • ライセンス：CreativeML Open RAIL++-M（商用利用可）
  • ベース：SDXL
  • 特徴：感情表現・ポーズの多様性が高く、スタンプ用途に最適。輪郭がはっきりしている
  • Civitai URL：https://civitai.com/models/261336
  • 作者：bluepen5805（blue_pencil-XL と同作者）　バージョンは v5.0.0 を選ぶ

2. Nova Anime XL（カラフル系スタンプ）¶

  • 用途：華やかでカラフルなキャラクター
  • ライセンス：CreativeML Open RAIL++-M（商用利用可）
  • ベース：SDXL
  • 特徴：2D・2.5D・3D的表現を使い分けられる。表情・衣装のバリエーションが豊富
  • Civitai：nova anime xl で検索 → Sort: Most Downloaded → 作者 AiAi のものを選ぶ

3. Flux.1 [schnell]（最新・高品質）¶

  • 用途：高品質イラスト全般
  • ライセンス：Apache 2.0（完全商用利用可）
  • 特徴：プロンプト指示への追従性が高く、細かい指定が通りやすい
  • 入手先：Hugging Face https://huggingface.co/black-forest-labs/FLUX.1-schnell
  • 注意：Flux.1 [dev] は商用利用不可。ファイル名に schnell と入っているものを選ぶ

PART 3：AUTOMATIC1111 基本設定¶

初回起動後の推奨設定¶

Settings（設定）タブで変更する項目：

Saving images/grids
  └ Images filename pattern: [date]-[seed]-[model_name]
  └ Save PNG info as sidecar .txt files: ON（プロンプトを記録）

User interface
  └ Localization: ja_JP（日本語化）※拡張機能のインストール後

Stable Diffusion
  └ Clip skip: 2（アニメ・イラスト系モデルの場合）

必須拡張機能のインストール¶

Extensions タブ → Install from URL

拡張機能 用途 URL
ControlNet ポーズ・構図の制御 https://github.com/Mikubill/sd-webui-controlnet
ADetailer 顔・手の自動修正 https://github.com/Bing-su/adetailer
LayerDiffuse 透過PNG生成（スタンプ用） https://github.com/layerdiffusion/sd-forge-layerdiffuse
Remove Background 背景自動除去 https://github.com/AUTOMATIC1111/stable-diffusion-webui-rembg
日本語化 UIを日本語に https://github.com/L4Ph/stable-diffusion-webui-localization-ja_JP

チェックリスト：Week 1〜2¶

Week 1：環境構築とモデルインストール¶

  • Stability Matrix をダウンロード・インストール
  • AUTOMATIC1111 を Stability Matrix 経由でインストール
  • 日本語化拡張をインストール
  • 商用利用可能なモデルを1つダウンロード（まず Anything XL 推奨）

Week 2：拡張機能と後処理の習得¶

  • ADetailer をインストール（顔崩れ自動修正）
  • Remove Background 拡張をインストール
  • Upscayl をインストール
  • 透過PNG を10枚作ってみる
  • Canvaで生成画像にテキストを入れてPOP風に仕上げてみる
