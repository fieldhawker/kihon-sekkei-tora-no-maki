在宅勤務男性向けランチプラン（筋肉維持ダイエット）¶

使用食材と栄養価¶

食材 量 カロリー タンパク質 炭水化物 脂質
玄米（生） 1合（180g） 約620kcal 約12g 約132g 約2g
納豆 1パック（45g） 約90kcal 約7.5g 約5g 約4g
卵 2個 約150kcal 約12.4g 約0.4g 約10g
合計 約860kcal 約32g 約137g 約16g

玄米1合は炊飯後 約330g（2膳分）になる

筋肉維持のポイント¶

  • タンパク質 32g はランチ1食として十分な量（体重70kg男性の1食目安は20〜30g）
  • 玄米は低GI食品のため血糖値の急上昇を抑え、脂肪蓄積を防ぐ
  • 納豆のビタミンK2と発酵菌は腸内環境を整え代謝をサポート
  • 卵の必須アミノ酸スコアは100（筋合成に最も効率的なタンパク源）

食べ方バリエーション（週5日プラン）¶

パターンA：納豆卵かけご飯（TKG納豆）¶

所要時間：5分 / 難易度：★☆☆

  1. 玄米を炊飯する
  2. 生卵2個を丼に割り入れる
  3. 納豆を混ぜてご飯にかける

ポイント： 最速。卵は生で食べることでビオチン損失を防げる。
注意： 生卵の食べすぎはビオチン欠乏リスクがあるため、このパターンは週2回まで。

パターンB：スクランブルエッグ + 納豆ご飯¶

所要時間：10分 / 難易度：★☆☆

  1. 卵2個をフライパンで半熟スクランブルエッグにする（油なし、テフロンパン）
  2. 納豆をご飯に混ぜる
  3. スクランブルエッグをご飯の横に盛り付ける

ポイント： 加熱卵はタンパク質の消化吸収率が生卵より高い（91% vs 51%）。

パターンC：納豆オムレツ丼¶

所要時間：15分 / 難易度：★★☆

  1. 卵2個を溶き、納豆を加えて混ぜる
  2. テフロンパンで薄焼きオムレツ状に焼く（半熟でたたむ）
  3. 玄米ご飯の上にのせる

ポイント： 納豆の糸が卵と絡み食べごたえ増。満腹感が高い。

パターンD：温泉卵 + 納豆ご飯¶

所要時間：15分（温泉卵作成含む） / 難易度：★★☆

温泉卵の作り方（鍋）： 1. 鍋に水1Lを沸騰させる 2. 火を止めて水200mlを加え温度を70〜75℃に下げる 3. 卵2個を入れてフタをして12〜13分放置 4. 冷水で1分冷やす

盛り付け： 1. ご飯に納豆をのせる 2. 温泉卵を割ってかける

ポイント： 黄身が半熟で栄養価と消化吸収のバランスが最も良い食べ方。

パターンE：玄米リゾット風（卵雑炊）¶

所要時間：15分 / 難易度：★★☆

  1. 炊いた玄米をフライパンに入れ、水100mlを加えて煮る
  2. 卵2個を溶き入れ、かき混ぜながら弱火で加熱（半熟状態で止める）
  3. 器に盛り、納豆をトッピング

ポイント： 食べやすく消化に優しい。体調不良時や食欲がない日に最適。

週間ローテーション例¶

曜日 パターン 理由
月 C（納豆オムレツ丼） 週始めは満腹感重視でモチベーション維持
火 B（スクランブルエッグ） タンパク吸収率を優先
水 D（温泉卵） 栄養バランス最適日
木 E（卵雑炊） 週中の疲れに優しく消化しやすく
金 A（TKG納豆） 最速で時短、週末前の業務集中日

食べる時間帯の推奨¶

  • 12:00〜13:00 に摂取するのが理想
  • 在宅勤務の場合、午前中に軽い筋トレ（スクワット・腕立て）を挟むと、筋肉へのタンパク質取り込み効率が上がる
  • 食後30分は横にならず、軽い散歩（10分程度）を推奨

ダイエット目標への適合性¶

指標 評価
カロリー（860kcal） 男性ランチとして適切（1食目安700〜900kcal）
タンパク質（32g） 筋肉維持に十分（◎）
低GI食品（玄米） 脂肪蓄積抑制（◎）
食物繊維（玄米・納豆） 腸内環境・代謝改善（○）
調理の手軽さ 在宅勤務の昼休み内で完結（◎）

想定体重減少ペース： この食事プランを継続し、朝・夕食も適切なカロリー管理を行えば、月2〜3kgの体重減少（うち筋肉量維持）が期待できる。
