# 設計書テンプレート集

> **このフォルダの目的**: 基本設計・詳細設計で作成する成果物の「すぐ使えるテンプレート」を提供します。テンプレートをコピーして使ってください。

---

## 使い方

1. 作りたい成果物に対応するテンプレートファイルを開く
2. ファイル全体をコピーして、プロジェクトの `docs/` フォルダに保存する
3. テンプレート内の `【】` で囲まれた部分をプロジェクトの内容に書き換える
4. `（例）` と書かれたサンプル行は削除して、実際のデータを入力する

## ファイル形式について

これらのテンプレートは **Markdown形式** で提供しています。

- Gitで管理でき、変更履歴が追跡できます
- VSCode・Obsidian・Notionなどで綺麗に表示できます
- **Excelが必要な場合**: 表部分をExcelにコピー＆ペーストすれば転用できます
- **Wordが必要な場合**: Pandocを使えばMarkdown→Word変換ができます

```bash
# PandocでMarkdown→Word変換
pandoc T-04_テーブル定義書.md -o テーブル定義書.docx
```
