## 画面遷移の概要

```
（認証前）
[ログイン画面] ─ 認証成功 ──→ [ダッシュボード]
             └ パスワード忘れ → [パスワードリセット申請] → メール送信 → [パスワード再設定]

（認証後 - 管理者）
[ダッシュボード] → [ユーザー一覧] → [ユーザー登録]
                                  → [ユーザー編集]

（認証後 - 一般ユーザー）
[ダッシュボード] → [商品一覧] → [商品詳細]
```

> 詳細な画面遷移は Figma の画面遷移図（B-06）を参照してください。
