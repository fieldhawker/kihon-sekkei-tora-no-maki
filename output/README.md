# システム開発 設計手順書 — 目次

> **この虎の巻について**: 要件定義完了から詳細設計完了まで、経験の浅いSEが迷わず進めるための設計手順・成果物ガイド・AI相談テンプレートをまとめた資料集です。
>
> **はじめて使う方へ**: まず [howto/00_概要と全体フロー.md](howto/00_概要と全体フロー.md) を通読してから、各フェーズのフォルダに進んでください。

---

## フォルダ構成

```
output/
├── README.md                        ← いまここ（全体目次）
├── howto/                           ← 全体の進め方（まずここを読む）
├── 基本設計手順/                     ← 基本設計フェーズのチェックリスト
├── 基本設計_成果物ガイド/             ← 基本設計で作る成果物のツール選定
├── 詳細設計手順/                     ← 詳細設計フェーズのチェックリスト
├── 詳細設計_成果物ガイド/             ← 詳細設計で作る成果物のツール選定
├── 共通ライブラリ/                   ← 詳細設計フェーズで先行して作る共通処理
├── 実装手順/                         ← 詳細設計→実装フェーズの移行手順
├── テスト実施手順/                   ← 単体・結合・システムテストの実施手順
├── 設計書テンプレート/               ← すぐ使える設計書テンプレート集（T-01〜T-08）
├── AI相談テンプレート/               ← フェーズ別AIへの相談テンプレート集
├── adr_sample/                      ← ADRのサンプルファイル（7件）
└── SES現場参画ガイド/               ← SES契約での客先常駐に特化した参考情報集
```
